# Focus Flow

A clean Pomodoro web timer built with HTML, CSS, and Vanilla JS. No frameworks – pure fundamentals.

## Why This Project?
- Demonstrates front-end basics done well
- Beautiful UI with circular progress
- Responsive, accessible, PWA-ready
- Sound notifications
- Dark mode toggle (coming)

Planned in 9 clean commits for portfolio.
## Installation
No install needed – open index.html in browser!

## Usage
- Click Start to begin 25-min Pomodoro.
- Pause/Reset as needed.
- Timer rings when done.
